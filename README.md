# 🔔 Distortionz Notify

A premium standalone **FiveM notification system** built for **Distortionz RP**.  
It provides a custom dark/red notification UI, smooth animations, status-based sounds, and easy exports for other resources.

---

## ✨ Features

- Premium dark glass UI
- Distortionz red/black styling
- Smooth slide-in / slide-out animations
- Progress timeout bar
- Queue handling
- Maximum notification limit
- Custom status types
- Per-status GTA frontend sounds
- Client-side export support
- Server-side event support
- Works with Qbox, QBCore bridge, and standalone scripts
- Can be used by any resource
